# Tomato Disease Detection

A Flutter-based mobile application that detects and identifies diseases in tomato plants using machine learning.

## Overview

This app uses computer vision and machine learning to analyze images of tomato plants and detect common diseases. It helps farmers and gardeners identify plant health issues early for better crop management.

## Features

- **Real-time Disease Detection** - Capture or upload tomato plant images for instant analysis
- **Disease Classification** - Identifies common tomato plant diseases
- **User-friendly Interface** - Easy-to-use Flutter mobile app
- **Quick Diagnosis** - Get disease information and management recommendations
- **Offline Capabilities** - Run inference on device for faster results

## Installation

### Prerequisites
- Flutter SDK installed
- Android/iOS development environment configured

### Steps

```bash
git clone https://github.com/HadiyaSalma27/Disease_Detector.git
cd Disease_Detector
flutter pub get
flutter run
```

## Usage

1. Open the app on your mobile device
2. Take a photo of your tomato plant or select an image from gallery
3. The app will analyze the image and display the diagnosis
4. View disease information and recommended treatments
